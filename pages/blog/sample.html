# 全局计算属性

在 VuePress 中，内置了一些核心的计算属性 (opens new window) (opens new window)，以供默认主题 (opens new window) 或自定义主题使用。

# # (opens new window)$site

这是你现在看到的这个网站的 $site 的值：

{
  "title": "VuePress",
  "description": "Vue 驱动的静态网站生成器",
  "base": "/",
  "pages": [
    {
      "lastUpdated": 1524027677000,
      "path": "/",
      "title": "VuePress",
      "frontmatter": {}
    },
    ...
  ]
}

# # (opens new window)$page

这是你现在看到的这个页面的 $page 的值：

{
  "title": "Global Computed",
  "frontmatter": {
    "sidebar": "auto"
  },
  "regularPath": "/zh/miscellaneous/global-computed.html",
  "key": "v-bc9a3e3f9692d",
  "path": "/zh/miscellaneous/global-computed.html",
  "headers": [
    {
      "level": 2,
      "title": "$site",
      "slug": "site"
    },
    {
      "level": 2,
      "title": "$page",
      "slug": "page"
    },
    ...
  ]
}

# # (opens new window)$frontmatter

$page (opens new window).frontmatter 的引用。

# # (opens new window)$lang

当前页面的语言，默认值为 en-US。

参考:

  • 多语言支持 (opens new window)

# # (opens new window)$localePath

当前页面的 locale 路径前缀，默认值为 /，当前页面为 /zh/。

参考:

  • 多语言支持 (opens new window)

# # (opens new window)$title

用于当前页面的 <title> 标签的值。

# # (opens new window)$description

用于当前页面的 <meta name="description" content="..."> 的 content 值。

# # (opens new window)$themeConfig

即 siteConfig.themeConfig。
